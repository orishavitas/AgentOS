# Integrations (v1)

AgentOS v1 is intentionally **repo-only**:
- No Cursor API integrations
- No subagent/router automation

## What “integration” means in v1
- Running the Node scripts locally:
  - `node scripts/new-project.js ...`
  - `node scripts/validate-scaffold.js ...`

## Future directions (document only)
- A router that issues file-scoped work orders and validates handoffs (see `docs/agent-os/07_future-subagents.md`)
